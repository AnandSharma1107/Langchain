# 🔎 Agentic Search App

A Streamlit-based AI agent that can search:
- 🌐 Web (DuckDuckGo)
- 📚 Wikipedia
- 📄 arXiv papers

Built using LangChain + Groq LLM.

---

## 🚀 Features

- Multi-tool agent (ReAct)
- Real-time streaming responses
- Chat interface
- External knowledge sources

---

## 🛠️ Tech Stack

- Streamlit
- LangChain
- Groq LLM
- DuckDuckGo Search
- Wikipedia API
- arXiv API
